
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter .

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Publish

```bash
# unit tests
$ npm run build

dist/shared/api-node-sdk (百度sdk package) 目录执行 npm install
根目录 pm2 start dist/main --name
pm2 stop name
pm2 start name
```
